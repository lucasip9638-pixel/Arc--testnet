3. Pressione Enter

========================================
  VERIFICAÇÃO
========================================

O servidor está rodando se você ver:
- Uma janela CMD/PowerShell aberta
- Mensagem: "Ready in X.Xs"
- Porta 3000 em LISTENING

Para verificar se está rodando:
1. Abra o Gerenciador de Tarefas (Ctrl+Shift+Esc)
2. Procure por processos "node.exe"
3. Se houver processos Node, o servidor está rodando

========================================
  PROBLEMAS COMUNS
========================================

1. "ERR_CONNECTION_REFUSED"
   → O servidor não está rodando
   → Execute: INICIAR-SERVIDOR-FINAL.bat

2. Página em branco
   → Aguarde alguns segundos (compilando)
   → Recarregue a página (F5)

3. Erro de compilação
   → Verifique a janela do servidor
   → Veja se há mensagens de erro em vermelho

========================================
  SOLUÇÃO DEFINITIVA
========================================

Execute este comando no terminal:

  cmd /c "npm run dev"

Depois:
1. Aguarde ver "Ready in X.Xs"
2. Abra o navegador
3. Acesse: http://localhost:3000
4. MANTENHA o terminal aberto!
